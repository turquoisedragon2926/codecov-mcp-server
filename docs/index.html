codecov-mcp

MIT

MCP server for Codecov. Query coverage data from Claude Code to check files, compare branches, and find untested code.

CODECOV_API_TOKEN=xxx npx codecov-mcp-server

Use Cases

Check coverage without leaving your editor.

Pre-push Checks

Ask Claude to check coverage before pushing. Catch missing tests early.

File Analysis

Get line-by-line coverage for any file. See exactly which lines need tests.

PR Review

Compare coverage between branches. Understand the impact of your changes.

Quick Start

Get your token and add to Claude Code config.

TOKEN
# Get API token https://app.codecov.io/account/access # Generate token # Copy for config below
CONFIG
# ~/.claude.json or project settings "mcpServers": { "codecov": { "command": "npx", "args": ["codecov-mcp-server"], "env": { "CODECOV_API_TOKEN": "..." } } }

Available Tools

8 tools for querying Codecov data.

Tool Description
codecov_get_coverage_totals Overall coverage metrics for a repository
codecov_get_file_coverage Line-by-line coverage for a specific file
codecov_get_coverage_tree Hierarchical coverage report by directory
codecov_compare_coverage Compare coverage between commits or PRs
codecov_list_repositories List repositories with coverage data
codecov_get_repository Detailed repository information
codecov_list_commits Commits with coverage data
codecov_list_pulls Pull requests with coverage info

Example Prompts

What to ask Claude Code.

PROMPTS
# Check repo coverage "What's the coverage for owner/repo?" # Check specific file "Which lines need tests in src/utils.ts?" # Compare branches "Compare coverage between main and my-branch" # Review PR "What's the coverage impact of PR #42?"
